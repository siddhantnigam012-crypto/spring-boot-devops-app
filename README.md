# Ubuntu Setup and Linux Practice

This repository contains my notes and practice work while learning Ubuntu and Linux fundamentals as part of my DevOps journey.

I created this setup to get comfortable with working in a Linux environment, since Linux is one of the most important skills for DevOps and Cloud roles.

## What I Practiced

* Installing Ubuntu Server using UTM
* Basic Linux commands
* Navigating files and directories
* Creating, deleting, and managing files
* Understanding file permissions
* User management basics
* Package installation using apt
* Working with the terminal
* Getting familiar with the Linux environment

## Why I Made This

Before starting with Docker, CI/CD, and other DevOps tools, I wanted to build a strong Linux foundation. This repository serves as a collection of my learning notes and hands-on practice.

## Tools Used

* Ubuntu Server 24.04 LTS
* UTM
* macOS
* Terminal

## What I Learned

One thing I realized while working on this setup is that Linux becomes much easier once you start using it daily. Initially, even simple commands felt confusing, but with regular practice I became more comfortable navigating the system and understanding how things work behind the scenes.

This project helped me gain confidence in using Linux, which will be useful as I continue learning Docker, Git, CI/CD, and Kubernetes.
